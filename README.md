# Para Transferi API Projesi

## Kullanılan Teknolojiler:
- .NET CORE WEB API
- Entity Framework Core
- Katmanlı Mimari
- .NET CORE MVC
- Microsoft SQL Server

## Operasyonlar:
- Müşteri Ol
- Giriş Yap
- Hesap Listesi
- Hesap Hareket Listesi
- Para Transferi
